### 책의 오탈자나 코드에 오류가 있는 경우 이슈를 남겨 주시거나 [이곳](https://tensorflowkorea.wordpress.com/%ED%85%90%EC%84%9C%ED%94%8C%EB%A1%9C-%EC%B2%AB%EA%B1%B8%EC%9D%8C/)에 글을 남겨 주세요.
